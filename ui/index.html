How it works
1 Paste one or more website addresses into the box above. Separate them by comma, space, or put each URL on a new line.
2 Click "▶ Start Scan". The scanner will crawl only inside each base URL.
3 When the scan finishes, a report will be generated and downloaded automatically.
When running the scanner locally, reports are saved in the:
PDF_Scan_Studio/scan_results/ folder
